
# MegaMonetkyBot — README (інструкція користування ботом)

Цей бот дозволяє користувачам:
- Натискати кнопку і заробляти монети
- Отримувати щоденні бонуси
- Купувати товари у магазині
- Запрошувати друзів через реферальне посилання
- Отримувати рекламу з AliExpress та Binance
- Грати у міні-гру "Вгадай число"
- Бачити прогрес і рівень
- Брати участь у лідерборді
- Адміну доступні розсилка і статистика

---

## Як запустити бота на Replit:

1. Перейди на [https://replit.com](https://replit.com)
2. Створи новий Python проект
3. Завантаж файл `mega_clicker_bot_PLUS_updated.py` у Replit
4. Натисни **"Run"**

---

## Як замінити токен бота:

У коді заміни рядок:
```python
TOKEN = "ТВОЙ_ТОКЕН_ТУТ"
```
на свій токен з [@BotFather](https://t.me/BotFather)

---

## Як встановити свій ADMIN_ID:
1. Запусти бота
2. Напиши `/getid`
3. Встав свій ID у змінну:
```python
ADMIN_ID = 123456789
```

---

## Команди для адміністратора:
- `/admin` — кількість користувачів
- `/top` — топ-10 багатіїв
- `/broadcast ТЕКСТ` — розсилка всім

---

## Як оновити рекламу AliExpress:
1. У коді знайди список `ali_links`
2. Додай або заміни посилання у форматі:
```python
ali_links = [
    "https://s.click.aliexpress.com/e/_Example1",
    "https://s.click.aliexpress.com/e/_Example2"
]
```

---

## Як змінити товари в магазині:
У блоці `if text == "Магазин":` ти можеш додати або редагувати товари.
